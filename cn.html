游戏项目

牙刷转转

个人独立开发的休闲小游戏。

  • 职责 单人设计与开发
  • 规则 来回滑动用鼠标滚轮或触控板滚动牙刷，在牙膏掉落前获得更多分数。
  • 游玩 WebGL 版本

Pave the Way

类俄罗斯方块无尽跑酷策略游戏，填充道路让小机器人保持前进。

  • 职责 游戏主设计，关卡设计，视觉设计
  • 规则 填充器每次射击增加一个地面方块，消除器每次射击移除一个方块，机器人停止超过五秒则游戏结束。填充器无法在红色方块上添加方块。消除器无法移除红/蓝色方块。
  • 下载 Windows 版本
  • 设计文档 查看PDF
  • 页面 项目页面（英文）

Crazy Pete’s Space Deals

商业模拟游戏，扮演二手宇宙飞船经销商，理解顾客的喜好并最大化利润。

  • 职责 UI 与 UX 设计，游戏美术，Unity UI 实现
  • 规则 向虚拟角色强调卖点，用零食维持他们的耐心值，每次成交后学习更优定价
  • 下载 Windows 版本
  • 页面 项目页面（英文）

Paint War

2D 大乱斗多人对战，像素风与 8 bit 音乐。通过角色技能与道具喷涂领地。

  • 职责 游戏美术
  • 输入 Nintendo Joycon 与键盘
  • 下载 Windows 版本
  • 页面 项目页面（英文）
